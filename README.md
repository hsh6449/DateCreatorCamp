### 3. 소감 
- 넷이 합을 맞춰본 경험이 처음인것 치고는 잘 맞았다고 생각한다.
- 하지만 논리적 근거를 마련하는데 어려움을 겪었고 역할분담에서 미흡한 모습을 보였다.
- 데이터를 이해하려고 초반 전처리와 도메인 조사로 역할을 나눈 점은 좋았음
